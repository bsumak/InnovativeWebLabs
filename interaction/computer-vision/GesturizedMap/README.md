# 🚨 Intervencijski Sistem – Zaščita & Reševanje

Napreden spletni sistem za vizualizacijo in upravljanje intervencij v realnem času, zasnovan za uporabo v štabih zaščite in reševanja.

## ✨ Funkcionalnosti

### 🗺️ Interaktivni zemljevid
- Prikaz intervencij na zemljevidu
- Barvno kodirani markerji
- Klik za podrobnosti

### 🎙 Glasovni vnos (VUI)
- Voden čarovnik (opis → lokacija → vrsta → potrditev)
- Podpora slovenščini
- Govorno vodenje (TTS)

### 🖐 Upravljanje z gestami
- ✊ premik zemljevida
- ✌ zoom
- ☝ izbor markerjev

### 📍 Geokodiranje
- OpenStreetMap (Nominatim)
- Prepoznavanje slovenskih krajev

### ➕ Ročni vnos
- Obrazec z autocomplete
- Validacija lokacije

## 🧰 Tehnologije
- HTML, CSS, JavaScript
- Leaflet.js
- MediaPipe Hands
- Web Speech API
