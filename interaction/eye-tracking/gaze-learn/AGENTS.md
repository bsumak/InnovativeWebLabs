# AGENTS.md

## Project goal
Build a polished production-ready educational web app called GazeLearn that uses eye tracking to adapt learning content.

## Engineering rules
- TypeScript only
- Clean modular architecture
- No giant files
- Prefer reusable components
- Accessibility and privacy are mandatory
- Eye tracking data stays local by default
- Fallback mode must work without camera permission
- No unfinished placeholder UI
- Production build, lint, and tests must pass

## UX rules
- Premium modern UI
- Reading-first layout
- Smooth but subtle motion
- Clear typography and spacing
- Focus mode must feel useful, not gimmicky

## Testing rules
- Add meaningful tests for heuristics and quiz personalization
- Add E2E coverage for main flows

## Docs rules
- Keep README and docs up to date with implementation
